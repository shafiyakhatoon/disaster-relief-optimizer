
 Disaster Relief Resource Allocation
### Greedy vs Dynamic Programming — Interactive Visualizer

---

 Project Overview

A web-based application that compares **Greedy Algorithm** and **Dynamic Programming (0/1 Knapsack)** to optimize the distribution of limited resources across disaster-affected regions.

---

 Features

- Greedy vs Dynamic Programming comparison  
- Hybrid approach (Greedy + DP)  
- Real-time region analysis  
- Add/remove custom regions  
- Impact-based allocation  
- Charts and execution trace  
- Winner display  

---

 Algorithms

### Greedy Algorithm
- Selects regions based on **impact per resource**
- Fast but not always optimal

### Dynamic Programming
- Uses **0/1 Knapsack**
- Always finds optimal solution

### Hybrid
- Combines speed + accuracy

---

 Impact Formula

```

Impact = (Urgency × Population) / Distance

```
